There is nobody on the project actively updating the translations. Any
received translations will be posted as patches.

2. All MySQL users must run ungrade script(s)

MySQL Users
-----------
This section only concerns users using MySQL to store BORG data. 

The SQL needed to transition from 1.5.1.2 to 1.5.2 is in file update152.sql.
The SQL needed to transition from 1.5.1 to 1.5.1.2 is in file update1512.sql.
The SQL needed to transition from 1.5 to 1.5.1 is in file update151.sql.
The SQL needed to transition from 1.4.x to 1.5 is in file update1_5.sql.

The borg.sql file has been updated with the SQL to create a new 1.5.2 database.


RUNNING
-------
For Windows: The installer can create shortcuts in the start menu and 
desktop (optional)
   Otherwise, double-click borg.jar from windows explorer.
   It is found in the install folder that you choose from the installer.

Command Line (any OS): The quickest way to run BORG from a command line 
is: java -jar borg.jar
   This will work as long as the java executable from the JRE is in your PATH
   on whatever OS you are running.
