Links and Attachments

 

Links are only available for HSQLDB and MySQL users. Links are not available for Remote Server users, even if the underlying remote database is MySQL or HSQLDB.

 

The following types of links can be added to appointments, tasks, projects, and addresses:

  • File Link – a link to a file somewhere in your file system
  • Attached File – a link to a file that BORG will copy into an attachment folder (supported for HSQLDB only). The attachment folder will be located in your HSQLDB database folder, so if you move your HSQLDB folder between machines (i.e. on a thumb drive), your attachments will always be available.
  • URL – a link to a URL that BORG will open in your default browser.
  • BORG Object – you can add a link to other BORG appointments, tasks, projects, addresses, and memos.

 

Links for an Appointment are seen in the appointment editor.

Links for a Task, Project or Address are seen on a new Links tab in their respective editors.

 

Links can be added, deleted, and opened.

 

Files and URLs are opened according to your operating system settings by launching the viewer specified. URLs will open in your default browser. When opening a file link, BORG will launch the specific viewer for that file type as specified in your OS settings (i.e. PDF viewer, XML viewer, Word, Excel, Open Office, etc…). If no viewer is specified, the file will not be opened.
