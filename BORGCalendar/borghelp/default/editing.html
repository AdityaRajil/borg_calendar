Editing Appointments

When the appointment editor first appears (after clicking the text area of a day), all of its fields are reset to default values and the text "***** NEW APPT *****" appears in the upper portion of the window. When this text appears, it means that the window is editing a new appointment that can be saved by hitting the Save button.

If the day being edited has existing appointments, they can be edited by clicking the corresponding appointment in the appointment list on the right side. The Save button saves any data for the appointment being edited.

Many of the fields in this window are intuitive and require no special documentation. Some items bearing explanation are:

·        If the "No specific time" checkbox is selected, no time string will appear in the main month display or print output and the appointment will be considered a non-scheduled appointment or "note".  Some calendar programs might call it an “all-day” appointment.

·        The duration pulldown is affects the size of the appointment in the week view and day view grids. The duration field does not specify the end time. It specifies the length of the appointment.

·        The color pulldown alters the appointment color on the main month view, and in the color printout. There is also an option (on the options screen) to sort by this color instead of only appointment time. This allows appointments to sort in the following color order – red, blue, green, black, white. White appointments are meant to be low priority and are intentionally less visible than other colors on the default background.

To Do's

If the To Do checkbox is selected, the appointment will be marked as a To Do. It will appear on the To Do list window, and can be marked as done from the To Do window also.

Repeating Appointments

The frequency pulldown allows you to set the frequency of an appointment that repeats. The times field should be set to the number of repeats desired.

Note that there are two types of monthly appointments:

· ones that repeat on the same date each month (i.e. the 6th of each month)

· ones that repeat on the same day and relative week each month (i.e. the third Thursday of each month). With this type, if no such day exists for a given month, the appointment is skipped for that month, but the repeat count is still decremented.

When deleting repeated appointments, Delete One Only and Delete buttons offer the choice of either deleting just the one occurrence of the repeat on the current day being edited, or all of the occurrences.

FYI - repeats are only shown on the calendar for 2 years from the present date. So  if  you repeat something yearly for 1000 years and try to look ahead to the year 3000, you will be disappointed.

Repeating To Do's

Like any other appointment, Repeating To Do's will appear multiple times on the month view (as determined by the frequency and Times). However, they will only appear once on the To Do list - starting with occurrence number one. If a repeating To Do is marked as done on the To Do list, the NEXT occurrence of the To Do will then be placed on the To Do list - this is done until the last occurrence has been marked as done.

Vacation, Half Day, Holidays

The vacation, half day, and holidays check boxes are simply used to mark the day in a different color on the main month view so that they stand out. They have no other effect. (half-day refers to a half-vacation day).

BORG currently can add certain US and Canadian holidays. This is an option from the preferences screen. BORG will also "guess" as to whether or not a US holiday should be marked using the holiday color or just normal day color. This is based on whether or not the holiday usually causes a day off from work in the US. So, for example, New Year's Day would be marked in weekend color, but Valentine's Day is not.

Whether or not you use the US holiday feature, if a certain day is a holiday for you, you can add an appointment with the name of the holiday and mark the Holiday checkbox. If you don’t like the default holidays, you can shut them off completely and enter your own.

Private Appointments

 Appointments marked as private using the private checkbox  only appear in the main month view and print output if the Options/Show Private checkbox is selected from the Options menu. In this way, Private appointments can be hidden when the calendar is used in a public setting. Example uses for private appointments would be to record certain doctor's appointments or to keep a log of your illnesses  or any other private non-work related items.
