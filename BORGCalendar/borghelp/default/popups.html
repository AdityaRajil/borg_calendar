Pop-Up Reminders 

As of release 1.5, you can choose the times for pop-up reminders to appear on  a per appointment basis.

 

For an appointment, at the chosen times, a reminder window will popup for the appointment and a sound will be played. If the reminder window is dismissed, it will reappear and the next chosen time – unless the option to suppress further reminders was chosen when the window was dismissed.

 

The set of selectable times of the popups are also tunable through the popups tab of the options screen. On this options screen, the set of times are in minutes. A negative number indicates a reminder that will go off after the appointment time.

 

There is an option to use the “system beep” sound for the popup. Use this if the default sound does not play properly.

 

Untimed appointments do not get popup reminders. As of release 1.5.2, any untimed todo’s for the current day will get popup reminders every half hour as long as any popup times are selected. If popup reminders are not wanted for untimed todos, then clear all the popup reminder times for the todo in the appointment editor.
