Importa/Exporta da XML

BORG può esportare/importare i suoi dati a/da file XML. Questa importazione/esportazione è completamente funzionante con un formato XML migliorato dalla release 1.2. L'azione di importazione/esportazione può essere trovata nella finestra del mese attraverso il menu Azione, e costituisce un buon modo per fare un backup dei vostri dati in una forma leggibile e modificabile umanamente.

L'importazione aggiungerà i dati dei file nell'attuale database, e questo può creare dei problemi, in quanto si potranno verificare delle duplicazione di argomenti, senza avere un controllo automatico; quindi abbiate accortezza in queste operazioni.

Dalla versione 1.3.2, è stata aggiunta l'importazione da un URL. Un esempio di URL potrebbe essere http://your.server.name/some/path/borg.xml. Questa funzionalità può essere utile quando si vogliono visualizzare degli appuntamenti sul web, che possono essere importati da diverse persone.

L'Esportazione verso un URL esporterà tutti e 3 i database come file borg.xml, mrdb.xml, addr.xml ovunque punti l'URL. L'esportazione non può essere fatta via http, ma evidentemente in altro modo, come per esempio via FTP. Per esempio, ftp://username@your.ftp.server/your/path è un URL con cui trasferire via FTP dei file XML su un server che non richiede una password. Se questa è richiesta, potete specificarla nell'URL - ftp://username:password@your.ftp.server/your/path.

Oltre che essere un modo per fare dei backup dei vostri dati, Importare/Esportare mediante un URL potrebbe essere un modo per condividere dati comuni, come meeting, vacanze, ecc..
