Icona Systray (solo per Windows)

BORG supporta l'iconda systray solo in sistemi Windows.

Affinchè l'icona systray funzioni correttamente, occorre che il file TrayIcon.dll si trovi nella directory di lavoro di BORG, che normalmente è la stessa che contiene il file borg.jar .

Il file trayicon.jar deve trovarsi nella directory "lib", all'interno della directory dove si trova il file borg.jar.

Il software trayicon è stato scritto da Jan Struyf, che non fa parte del team di BORG.
