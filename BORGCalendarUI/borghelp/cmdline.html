Command Line Options

To run from the command line, type

 java –jar borg.jar <command line options>

-db <dir>

set database directory

-trayname <string>

set tool tip text for system tray icon ( windows )

 

Both options listed above aid in starting multiple Borg’s on the same machine with different databases. This will work, but isn’t really recommended.

 

There are other obsolete command line options that are no longer supported.
