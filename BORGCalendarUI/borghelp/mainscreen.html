The Calendar

The BORG main screen shows a single month at a time. The Navigation buttons appear on the bottom of the screen and are intuitive (Let me define intuitive. When I say intuitive, it means that I am describing something that I haven’t documented yet, and might not document, because it seems to be obvious enough. For example the "next" button on the main month window).

If the current month is being viewed, the current day will appear dark pink. Weekends are shaded slightly darker than weekdays. Days marked as holidays appear the same color as weekends. Days can also be marked as vacation days, and if so, will appear green.

To add/remove/update appointments for a given day, just click on the text area for that day.

Week View

To the right of each week is a button that will produce a "week view" when pressed. The week view is not interactive. It is a grid view of the week that plots the scheduled appointments for each day according to their start time and duration. The start and end range of the grid is user tunable through the options screen. The week view can be sent to a printer via a menu option.

 

Day View

A printable view of a single day’s appointments can be obtained by pressing the date button for a day. Like the week view the day view is not interactive. It can only be viewed or printed. The start and end hours of the week view will also be used as the start and end hours for the day view.
